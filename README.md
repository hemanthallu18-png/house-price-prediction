# House Price Prediction

Internship project (Week 1) — a regression model that predicts house prices based on
property features (size, rooms, location, age, etc.) and identifies which features
most strongly influence price.

## Contents

- `analysis.ipynb` — full notebook: data loading, cleaning, model building (Linear
  Regression & Random Forest), evaluation, and visualizations
- `Housing.csv` — dataset used ([source](https://www.kaggle.com/datasets/yasserh/housing-prices-dataset))
- `summary.pdf` — 1-page written summary of findings
- `chart1_price_distribution.png` — distribution of house prices
- `chart2_correlation_heatmap.png` — feature correlation heatmap
- `chart3_actual_vs_predicted.png` — actual vs. predicted price
- `chart4_feature_importance.png` — feature importance

## Tools

Python, Pandas, Scikit-learn, Matplotlib/Seaborn, Jupyter Notebook

## Models

- Linear Regression
- Random Forest Regressor

Evaluated using MAE, RMSE, and R² Score.
